# Новый Мир

Стартовый workspace для платформы ПК "Новый Мир".

## Что уже подготовлено

- монорепо на npm workspaces;
- фронтенд на Next.js с App Router;
- базовая мультиязычность RU / UKR / ENG;
- стартовая главная страница с акцентом на простой UX;
- заготовка пакета CMS под Strapi + PostgreSQL;
- корневые переменные окружения для локального запуска.

## Структура

- `apps/web` — публичный сайт и клиентская часть платформы;
- `apps/cms` — пакет для CMS и админ-панели;
- `.github/copilot-instructions.md` — состояние настройки workspace.

## Быстрый старт

1. Установить зависимости фронтенда:
   `npm install --workspace @novyi-mir/web`
2. Запустить сайт:
   `npm run dev`
3. Открыть `http://localhost:3000`

## Что покрывает стартовый фронтенд

- переключение языка между RU / UKR / ENG;
- посадочная страница в стиле внешнего дизайна с адаптивностью и анимациями;
- первичная структура под каталог, заказы, точки выдачи и контентные страницы;
- готовность к подключению headless CMS.

## Локальная админ-вкладка Home

- адрес: `/ru/admin`, `/uk/admin`, `/en/admin`;
- редактируются SEO, текст и изображения каждого блока главной страницы;
- сохранение выполняется через API `GET/PUT/DELETE /api/home-content/[locale]`;
- данные хранятся в PostgreSQL (таблица `home_content`, поле `content` типа `jsonb`).

## CMS

В `apps/cms` создан bootstrap-пакет и документация по следующему шагу. Strapi CLI в текущем окружении не был полностью инициализирован автоматически, поэтому генерация полноценной CMS вынесена в отдельный шаг после подтверждения локальной БД и желаемой схемы хранения медиа.
